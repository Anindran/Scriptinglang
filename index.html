SCRIPTING LANGUAGES VS PROGRAMMING LANGUAGES

Every scripting language is basically a programming language. The only theoretical difference is that a scripting language does not include the compilation step- it is interpreted instead. For instance, one needs to first compile a C program before running it. On the other hand, one does not need to compile a scripting language such as PHP or JavaScript
